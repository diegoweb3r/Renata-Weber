<h1 align="center"> 🎭 RENATA WEBER - Portfólio Artístico </h1> <h2 align="center"> 📖 Descrição </h2> <p align="justify"> O <strong>Portfólio Renata Weber</strong> é uma aplicação web desenvolvida para apresentar de forma profissional e impactante a trajetória, habilidades e trabalhos de uma atriz. O projeto tem como foco destacar a identidade artística, reunindo fotos, currículo e formas de contato em uma experiência moderna, fluida e acessível. </p> <p align="justify"> Este projeto foi concebido com atenção especial à estética visual e à experiência do usuário, utilizando design responsivo e uma arquitetura de CSS modular que garante escalabilidade e fácil manutenção. </p>

---

### 🛠 Tecnologias Utilizadas  
- **Estruturação:** HTML5 Semântico para acessibilidade e SEO.
- **Estilização:** CSS3 Moderno utilizando Metodologia Modular e Custom Properties (Variáveis CSS).
- **Layout:** CSS Grid e Flexbox para responsividade intrínseca (sem dependência excessiva de Media Queries).
- **Tipografia:** Tipografia moderna para uma interface contemporânea e legível.
- **Versionamento:** Git & GitHub para controle de fluxo e documentação.
